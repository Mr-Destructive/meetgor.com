## Commentary

- As per this and more experimentation, it could be said Spark is the most competent and well engineered AI Assisted Code Environment.
- Why? Because it’s GitHub, it has integrated the LLM in such a way that it seamlessly blends with GitHub Actions, Pages, Repositories, Secrets and Environments, Google had a great chance with FireStudio, but failed there.
- GitHub seems to have taken a better and more engineered approach and not rushed it.
